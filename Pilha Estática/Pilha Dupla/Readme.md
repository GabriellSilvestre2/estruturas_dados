Implemente uma aplicação de pilhas duplas onde será necessário empilhar e desempilhar elementos de duas categorias diferentes:
uma pilha de números inteiros e uma pilha de caracteres. A aplicação deve fornecer um menu interativo no console para realizar 
as seguintes operações:

1- Empilhar na Pilha 1 (números inteiros): O usuário deve poder inserir um número inteiro para ser empilhado na primeira pilha.

2- Empilhar na Pilha 2 (caracteres): O usuário deve poder inserir um caractere para ser empilhado na segunda pilha.

3- Desempilhar da Pilha 1: O número no topo da primeira pilha (números inteiros) deve ser removido e exibido.

4- Desempilhar da Pilha 2: O caractere no topo da segunda pilha deve ser removido e exibido.

5- Exibir as duas pilhas: Exibir os elementos de ambas as pilhas sem modificá-las.

6- Encerrar o programa.
